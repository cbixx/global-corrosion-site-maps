# Corrosion Atlas icon pack

Final direction:
- thin navy circular atlas boundary
- organized site-dot field
- rust-orange angular crack/front slightly right of center
- one modest branch
- transparent favicon mark plus app-icon tiles

Recommended files to copy to the project root:
- favicon.ico
- favicon.svg
- apple-touch-icon.png
- icon-192.png
- icon-512.png
- site.webmanifest

Optional brand assets:
- corrosion-atlas-mark.svg
- corrosion-atlas-mark-tile.svg
- corrosion-atlas-mark-dark.svg
- corrosion-atlas-mark-rust.svg
- corrosion-atlas-logo-horizontal.svg
- icon-512-dark.png
- icon-512-rust.png

Add this inside the <head> of index.html:

<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" href="/favicon.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
<meta name="theme-color" content="#10293A">

If you later create a separate landing page, use corrosion-atlas-logo-horizontal.svg for the header.
