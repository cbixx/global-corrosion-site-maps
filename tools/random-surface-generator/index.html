How the generator works

The generated coating thickness field is represented as:

h(x,y) = h0 + Δh(x,y)

The tool first creates a seeded random field. The random field is transformed into the frequency domain using a fast Fourier transform. An anisotropic Gaussian spectral filter is then applied using the correlation lengths ℓx and ℓy. This creates an anisotropic surface pattern.

α = ℓx / ℓy

After inverse transformation, the fluctuation field is shifted to zero mean and rescaled so that its root-mean-square value approximately matches the target roughness Rq

η = Rq / h0

Here, α is the anisotropy ratio and η is normalized roughness. Large η values can generate unrealistically thin or negative local coating thicknesses, so the tool checks the generated field before enabling export.

The exported CSV and TXT files use SI units. Coordinates x, y, and coating thickness h are written in metres.
